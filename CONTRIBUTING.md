# Contributing to CloudInk

Thanks for your interest in contributing to CloudInk! Here's how you can help.

## How to Contribute

1. **Fork** the repository
2. **Create a branch** for your feature or fix: `git checkout -b feature/my-feature`
3. **Make your changes** and test them locally with `hugo server`
4. **Commit** with a clear message describing the change
5. **Push** your branch and open a **Pull Request**

## Development Setup

```bash
# Clone the repo
git clone https://github.com/pescarcena/CloudInk.git

# The theme is designed to work with Hugo extended edition
hugo version  # Requires v0.128.0+

# Run the example site (if available) or test with your own Hugo project
hugo server
```

## Guidelines

- Keep changes focused and minimal
- Test with both light and dark themes
- Ensure responsive design works on mobile
- Follow existing code style and conventions
- Update documentation if needed

## Reporting Bugs

Open an [issue](https://github.com/pescarcena/CloudInk/issues) with:

- A clear description of the bug
- Steps to reproduce
- Expected vs actual behavior
- Hugo version and OS
