# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in CloudInk, please report it responsibly:

1. **Do not** open a public issue
2. Email **paul.escarcena@gmail.com** with details of the vulnerability
3. Include steps to reproduce if possible

You can expect an initial response within 48 hours. We will work with you to understand and address the issue before any public disclosure.
